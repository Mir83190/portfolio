Mobilité du personnel de MTPM , SNCF-Réseau Mistral

Horaires Réseau Mistral

Contexte

Développement de scripts Python exploitant l’API SNCF et l’API SIRI du réseau Mistral, dans le but de faciliter l’accès aux horaires pour le personnel. Deux tableaux de bord ont été créés et diffusés sur les écrans du siège afin de rendre l’information accessible au plus grand nombre.

Technologies utilisées

Python (script de calcul)
API SNCF (json)
API SIRI (xml)
Request & Post

Résultats obtenus

✓
Plus de 300 consultations ont été enregistrées sur chaque tableau de bord par le personnel de TPM.
✓
Une meilleure accessibilité aux horaires en temps réel a été assurée pour l’ensemble des agents grâce à une interface claire et centralisée.
✓
La diffusion sur les écrans du siège a permis une visibilité maximale et un gain de temps pour l'ensemble des équipes.
✓
Le projet a renforcé la communication interne en centralisant des données clés dans un format visuel simple et accessible.
